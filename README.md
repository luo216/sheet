## 配置基本环境

- linux环境下直接执行以下命令,前提你已经安装了python3
```shell
./setup.sh
```

## config.yaml

- 该项目暂时由yaml来配置，后续会支持json
- 通过yaml来设置 字段名, 数据其实位置, 数据长度, 可视化图像类型

## 如何启动

- liux环境下直接执行以下命令
```shell
./start.sh
```
